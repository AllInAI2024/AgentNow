# Hermes 系统管理模块设计方案

> 版本：v2.0  
> 日期：2026-04-28  
> 适用：AgentNow 企业智能体平台

---

## 一、设计目标

### 1.1 目标概述

为 AgentNow 超级管理员提供一个完整的 Hermes 系统管理功能模块，包括系统监控、对话日志管控、Profile 管理、技能管理、配置管理等功能。

### 1.2 设计原则

| 原则 | 说明 |
|------|------|
| **集中管控** | 超级管理员可查看和管理所有用户的 Hermes 相关数据 |
| **权限隔离** | 普通用户无权限访问此模块，仅超级管理员可见 |
| **分级菜单** | Hermes 作为一级菜单，各功能作为二级菜单 |
| **完整审计** | 所有操作记录审计日志，可追溯 |
| **企业级管控** | 对话日志完整可查，无隐私限制（企业内部系统） |

### 1.3 权限设计

| 权限标识 | 权限名称 | 说明 |
|----------|----------|------|
| `hermes:view` | Hermes 查看权限 | 可查看所有 Hermes 相关页面 |
| `hermes:admin` | Hermes 管理权限 | 可执行管理操作（重启、停止、删除等） |
| `hermes:conversation:view` | 对话日志查看 | 可查看所有对话日志 |
| `hermes:conversation:export` | 对话日志导出 | 可导出对话日志 |
| `hermes:conversation:delete` | 对话日志删除 | 可删除对话日志 |

#### 对话日志数据结构

```typescript
interface Conversation {
  id: string                    // 会话ID
  profile_name: string          // Profile 名称
  user_id: number               // 用户ID
  user_name: string             // 用户名
  start_time: string            // 开始时间
  last_message_time: string     // 最后消息时间
  message_count: number         // 消息总数
  user_message_count: number    // 用户消息数
  ai_message_count: number      // AI 消息数
  tool_call_count: number       // 工具调用次数
  status: 'active' | 'completed' // 状态
  messages: Message[]           // 消息列表
}

interface Message {
  id: string
  role: 'user' | 'assistant' | 'system' | 'tool'
  content: string
  timestamp: string
  tool_calls?: ToolCall[]       // 工具调用（仅 AI 消息）
  tool_result?: ToolResult      // 工具结果（仅 tool 消息）
}

interface ToolCall {
  id: string
  name: string                  // 工具名称
  arguments: Record<string, unknown> // 参数
}

interface ToolResult {
  tool_call_id: string
  output: string                // 执行结果
  is_error: boolean             // 是否出错
}
```

#### 搜索和筛选条件

| 条件 | 类型 | 说明 |
|------|------|------|
| 关键词 | 文本 | 在对话内容中搜索 |
| 用户 | 下拉选择 | 按用户筛选 |
| Profile | 下拉选择 | 按 Profile 筛选 |
| 时间范围 | 日期选择器 | 开始时间和结束时间 |
| 状态 | 下拉选择 | 全部/进行中/已完成 |
| 消息数范围 | 数字输入 | 最小消息数、最大消息数 |
| 是否有工具调用 | 复选框 | 仅显示有工具调用的对话 |

def quicksort(arr):
    if len(arr) <= 1:
        return arr
    pivot = arr[len(arr) // 2]
    left = [x for x in arr if x < pivot]
    middle = [x for x in arr if x == pivot]
    right = [x for x in arr if x > pivot]
    return quicksort(left) + middle + quicksort(right)
